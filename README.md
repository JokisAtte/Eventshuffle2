## Description

Eventshuffle 2 backend API

Built with [NestJS](https://github.com/nestjs/nest) framework.

## Installation
Navigate to the cloned repository

```bash
$ npm install
```

## Running the app

```bash
$ docker-compose -f "docker-compose.yml" up -d --build
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
